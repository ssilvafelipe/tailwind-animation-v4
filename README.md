# Tailwind 4 Animations Plugin

Plugin basado en [midudev/tailwind-animations](https://github.com/midudev/tailwind-animations), que añade utilidades de animaciones a TailwindCSS 4 de forma sencilla y personalizable.

## 🚀 Instalación

Asegúrate de tener [pnpm](https://pnpm.io/) instalado.

```bash
pnpm install
```

## ⚙️ Scripts disponibles

- **Generar CSS**  
  ```bash
  pnpm run generate:css
  ```
  Genera el archivo CSS con las animaciones.

- **Build completo**  
  ```bash
  pnpm run build
  ```
  Compila el plugin para producción.

- **Build solo CSS**  
  ```bash
  pnpm run build:css
  ```
  Compila únicamente el CSS de las animaciones.

- **Debug CSS**  
  ```bash
  pnpm run debug:css
  ```
  Muestra el output del CSS generado para depuración.

- **Tests**  
  ```bash
  pnpm run test
  ```
  Ejecuta la suite de pruebas.
